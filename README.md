# SaaS Revenue Forecasting Model

## Project Overview
A complete end-to-end financial analysis project 
built using SQL, Excel and Power BI on real-world 
telecom customer data (7,043 records).

## Tools Used
- PostgreSQL — Database & SQL Analysis
- Microsoft Excel — Financial Modelling
- Power BI — Interactive Dashboard

## Dataset
- Source: Kaggle (Telco Customer Churn)
- Records: 7,043 customers
- Features: 21 columns including contract type, 
  charges, churn status, internet service

## Key Findings
- Total ARR: $5.47M across 3 contract types
- Month-to-month churn rate: 42.71% 
- $1.67M ARR lost annually to churn
- Fiber optic + Electronic check = 53% churn rate
- $892,156 in recoverable ARR identified

## Project Structure

### 1. SQL Analysis
- ARR by contract type
- Churn rate analysis
- MRR waterfall by tenure
- Revenue by segment
- Customer health scoring

### 2. Excel Financial Model
- 3-Statement Model (P&L, Balance Sheet, Cash Flow)
- 24-month revenue forecast
- Monte Carlo simulation (1,000 runs)
- Scenario analysis (Base/Bull/Bear)

### 3. Power BI Dashboard
- Executive Summary
- Churn Analysis
- Customer Segmentation
- Forecast & Scenarios
